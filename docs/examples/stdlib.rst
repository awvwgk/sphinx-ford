Fortran Standard Library (stdlib)
=================================

`Fortran stdlib <https://github.com/fortran-lang/stdlib>`_ is a
community-driven standard library for modern Fortran, providing
utilities for strings, sorting, statistics, linear algebra, and more.

The documentation below is generated from the stdlib v0.8.1 source
code using ``f:automodule``. stdlib uses fypp preprocessing. The FORD
bridge processes the preprocessed output automatically.
