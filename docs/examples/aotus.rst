AOTUS: Lua scripting for Fortran
================================

`AOTUS <https://github.com/apes-suite/aotus>`_ provides a Fortran
wrapper around the C-API of the Lua scripting language, allowing
Lua scripts to be used as configuration files in Fortran applications.
