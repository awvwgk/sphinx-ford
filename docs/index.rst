sphinx-ford
===========

A Sphinx domain for Fortran with a FORD bridge for automatic
documentation of Fortran source code.

sphinx-ford provides a Fortran language domain (``f``) for Sphinx,
enabling you to document Fortran modules, procedures, types, and other
entities directly in reStructuredText. It also includes a bridge to
`FORD <https://github.com/Fortran-FOSS-Programmers/ford>`_ for
auto-documenting Fortran source code.
